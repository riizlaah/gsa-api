# GSA (GrowthSeekerAcademy) API
REST API untuk online courses
> Base URL : `address:port/gsa-api/v1/`

## Features

- Basic Auth (login/register)
- Get courses with pagination
- Get course details
- Purchase courses with coupon
- CRUD Coupon (admin)
- CRUD Courses (admin)
- Get Transactions
